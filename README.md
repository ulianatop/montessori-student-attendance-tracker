To run the node.js server:
1. Start the Apache and SQL servers in XAMMP
2. Have the main project folder (CSD350-project) open in vscode.
3. In the vscode console, put "node NodeServer/server.js

To run the react server:
1. In file explorer, navigate to CSD350-project/front_end/react/attendance
2. In the address bar, put "cmd", that opens a command prompt in that file
3. In the prompt, put "npm run dev"
